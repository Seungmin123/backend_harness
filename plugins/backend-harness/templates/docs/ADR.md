# ADR (Architecture Decision Records)

> **append-only**. 기존 항목은 수정하지 않는다 — 결정이 뒤집혔으면 새 ADR을 추가해 이전 항목을
> "Superseded by ADR-N"으로 표시한다. 번호 충돌은 머지 시 조정한다.
> `CLAUDE.md`의 "협업 규칙" 섹션 참조.

## 작성 형식

```
## ADR-{N}: {결정 제목}

- **날짜**: YYYY-MM-DD
- **상태**: Accepted | Superseded by ADR-{M}
- **배경**: 어떤 문제/제약 때문에 이 결정이 필요했는가
- **결정**: 무엇을 하기로 했는가
- **대안**: 검토했던 다른 선택지와 기각 이유
- **결과**: 이 결정으로 감수하는 트레이드오프
```

---

(아직 기록된 ADR 없음 — 새 기술 결정이 생기면 위 형식으로 추가)
